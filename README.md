# Vistoria Aero Rancho - PWA

Aplicativo web instalável no celular para fiscalização operacional.

## Rodar no computador

```bash
npm install
npm run dev
```

## Gerar versão de produção

```bash
npm run build
```

## Publicar no Vercel

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para o repositório.
3. Entre no Vercel e escolha **Add New Project**.
4. Selecione o repositório do GitHub.
5. Framework: **Vite**.
6. Clique em **Deploy**.

Depois de publicado, abra o link no celular e escolha **Adicionar à tela inicial**.
